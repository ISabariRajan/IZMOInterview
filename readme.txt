The Program to analyze is written in main.py written in 3.11
Python version and it uses the following libraries:
-   pandas
-   pylint

Setup dev environment
---------------------------
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py <Input File path>


Output
-------
The output of this program will be multiple CSV files named "subnet_security-group.csv, security-group.csv, subnet.csv"
which contains the analysis results for each input file provided as an argument.
